Pastry Design Language

Warm Tactile Glass for the clipboard panel.

这张画板把当前设置页已经形成的语言整合进来，再延展到面板页面： 深色磨砂托盘、暖白纸片卡片、焦糖金实体控件，以及来自复制来源的主题色。

设计逻辑卡片

1. 外层像托盘 面板是临时呼出的 tray，用深色磨砂玻璃承载所有内容。
2. 卡片像纸片 剪贴板内容是暖白纸片，降低系统白的冰冷感。
3. 控件像实体按钮 搜索、筛选、切换、设置都要有轻微按压和内高光。
4. 强调只用焦糖金 选中、快捷键编号、主状态统一用 Pastry 的烘焙金。

1. 外层托盘

使用设置页侧栏同源的深色磨砂材质。边缘要软，颜色偏深，不要变成亮白玻璃。

2. 顶部工具

搜索、筛选、All/Favorites、设置入口统一成小型实体控件。选中态用焦糖金。

3. 剪贴板卡片

卡片是暖白纸片，顶部保留应用主题色条和应用徽章，内容区保持高可读性。

4. 状态语言

普通、选中、危险三种状态要靠边线和材质变化表达，避免大面积强色块。

实现规则

面板优先扫视 不要把设置页的信息密度直接搬过来；面板是快速选择，不是管理页面。
玻璃必须压暗 深色背景上的玻璃应更暗、更柔，边缘不要白到突兀。
卡片保留来源感 应用主题色和应用图标仍是 Pastry 的核心识别，不要抹掉。
按钮保持拟物 所有高频操作都要有按压、内高光、短阴影，和设置页控件一致。
